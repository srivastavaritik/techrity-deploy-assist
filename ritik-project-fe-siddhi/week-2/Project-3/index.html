HTML Tags Description

Tag Inline / Block Description
<div> Block It is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.
<a> Inline It defines a hyperlink, which is used to link from one page to another.
<main> Block It specifies the main content of a document.
<form> Block It is used to create an HTML form for user input.
<span> Inline It is an inline container used to mark up a part of a text, or a part of a document.
<button> Inline It defines a clickable button.
<header> Block It represents a container for introductory content or a set of navigational links.
<input> Inline It specifies an input field where the user can enter data.
<p> Block It defines a paragraph.
<li> Block It defines a list item.
<img> Inline It is used to embed an image in an HTML page.
<br> Inline It inserts a single line break.
<footer> Block It defines a footer for a document or section.
<textarea> Inline It defines a multi-line text input control.
<time> Inline It defines a specific time.
