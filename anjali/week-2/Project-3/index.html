HTML Tags

Tag Type Description
<div> Block This tag is used as a container for HTML elements.
<p> Block This tag defines a paragraph.
<a> Inline This tag defines a hyperlink, which is used to link from one page to another.
<img> Inline This tag is used to embed an image in an HTML page.
<main> Block This tag specifies the main content of a document.
<form> Block This tag is used to create an HTML form for user input.
<span> Inline This tag is an inline container used to mark up a part of a text, or a part of a document.
<button> Inline This tag defines a clickable button.
<header> Block This tag represents a container for introductory content or a set of navigational links.
<footer> Block This tag defines a footer for a document or section.
<input> Inline This tag specifies an input field where the user can enter data.
<br> Inline This tag inserts a single line break.
<Table> Block This tag defines an HTML table.
<li> Block This tag defines a list item.
<textarea> Inline This tag defines a multi-line text input control.
<time> Inline This tag defines a specific time.
